用户许可协议

一、关于App    

    1.1  请您悉知，上海应用技术大学学生助手App(以下简称SITschool)并非官方发布和许可的应用，SITschool目前由我个人开发维护。

    1.2  鉴于您在使用SITschool的服务时，应用可能会收集和使用您的相关信息，希望您能谅解并允许许可协议。

    1.3  本App目前仅通过酷安助手，PP助手，豌豆荚，及App发布页发布，请不要在其他渠道下载，可以通过App内部的“检查更新”进行更新。

    1.4  用户在使用本App提供的服务时，除遵守“用户许可协议外”，还应遵守相关法律法规，维护社会公共利益。

    1.5  用户在“社区”所分享的信息，应当没有版权纠纷，如发现版权问题，应及时联系并下架处理。用户发布的内容所带来的不良后果， SITschool在必要时有权进行处理。

 

二、SITschool可能会收集的信息

    2.1  本软件通过对学校OA网页信息抓取、分析，来完成用户所需要的信息和功能。所以在登录后会保存您的账号密码信息。

    2.2  注册信息，在使用SITschool服务时，向App提供的个人信息，如：联系方式，姓名，icon等。

    2.3  其他方向SITschool分享的信息，您可以使用其他App分享资讯到上应社区。

    2.4  SITschool不会向用户收集手机号，银行卡号，身份证号等涉及个人隐私安全的相关信息。请您在进行相关操作时仔细斟酌。

 

三、SITschool可能使用的权限

    3.1  为了提升用户体验，SITschool集成了“极光推送”服务，在使用中可能的权限：睡眠锁，设备信息。另外运行App还需要的基本权限：存储权限，网络权限，Wifi权限。

    3.2  在使用相关服务中，可能会通过GPS 和WiFi收集您的地理位置信息，例如个性化的资讯提供服务。您可以通过关闭位置共享停止该服务。

 

四、信息政策

    4.1  SITschool使用各种安全技术和手段，尽可能保证您的信息安全，防止丢失、不当使用未经授权的披露等。

    4.2  SITschool可能在一下情况对您的相关信息进行处理：

         （a）遵守适用的法律法规，遵守相关法院或法律程序的要求。

         （b）遵守相关政府的要求。

         （c）遵守学校相关规定。

         （d）SITschool为保护其他用户的信息安全和合法权益必须的用途。

 

（如需要联系和反馈，请通过App内部的反馈渠道联系作者）
